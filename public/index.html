Popcorn Dynamic Build Tool

This build tool uses a new node app I've developed. The app takes a query string and builds you a custom version of popcorn based on it.

You can get the source for the node app here.

The app uses Uglify.js to minify code, and caches requests for up to 5 days, so that similar requests can be served faster (This demo doesn't have the cache ability enabled - no mongodb available on nodejitsu).

To use this tool, select the bits of popcorn you'd like, and also select if you want minified( Production ) code, or unminified ( Development ) code.

When you're ready, click the get URL button near the bottom to generate your link. The sample response area will be populated with the actual respose the app will give you ( and caches it, as a bonus :P ). You can now copy that link and use it in a script tag to always* get your desired build.

* This is just a TEST page, please do not use the generated links for anything, demo or production, as the endpoint for the app is not permanent.
